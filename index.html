📚 Materi Pembelajaran Dasar Web & JavaScript

1. HTML (HyperText Markup Language)

HTML adalah kerangka dasar setiap halaman web. Menggunakan elemen seperti heading, paragraf, gambar, dan tautan. Contoh: <h1>Judul</h1>

2. CSS (Cascading Style Sheets)

CSS digunakan untuk mempercantik tampilan: warna, layout, font, dan efek animasi. Flexbox dan Grid adalah teknik modern untuk tata letak.

3. JavaScript (JS)

JS membuat halaman menjadi interaktif: merespon klik, memvalidasi form, manipulasi DOM, hingga membuat kuis seperti yang ada di web ini!

🌟 Tips Belajar

Praktik langsung dengan menulis kode, kerjakan kuis, dan eksplorasi proyek kecil. Konsistensi adalah kunci!
